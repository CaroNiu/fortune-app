# fortune-app

每日运势小程序 - 微信云开发版本

## 功能特点

- 🎯 基于生日计算每日运势
- 📊 四维运势：事业、财运、感情、健康
- ✅ 每日打卡，记录运势
- 📅 打卡日历，查看连续打卡
- 📈 统计分析，平均运势

## 技术栈

- 微信小程序
- 微信云开发（云数据库）
- 前端直接计算运势算法

## 部署步骤

1. 申请微信小程序账号
2. 开通微信云开发
3. 创建数据库集合：users、checkins
4. 下载微信开发者工具导入项目
5. 在 app.js 中填入云开发环境 ID
6. 上传发布

## 免费额度

- 云函数调用：500万次/月
- 云数据库读：5万次/天
- 云数据库写：3万次/天
- 足够日活 1000+ 用户使用
